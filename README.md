# AI Leaderboard Skill

📊 OpenClaw skill for fetching AI model leaderboard rankings.

## Overview

Fetches AI model leaderboard data and displays rankings in formatted output.

## Usage

When user mentions: `AI榜单`, `模型排行`, `LLM排名`

## Features

- AI model rankings data
- Performance comparison
- Feishu card output support

## Requirements

- OpenClaw agent
- Web access

## Installation

This skill is managed by OpenClaw. Install via:
```bash
openclaw skills install ai-leaderboard
```
